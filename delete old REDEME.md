# 🏠 PGLife - PG Accommodation Finder Web Application


PGLife is a web-based application that helps students and working professionals find suitable Paying Guest (PG) accommodations in different cities.

The platform allows users to browse PG properties, apply filters, view detailed property information, register and log in to their accounts, mark properties as interested, and manage their profiles through a personalized dashboard.

This project was developed using PHP, MySQL, JavaScript, Bootstrap, HTML, and CSS, with REST API integration for dynamic user interactions.


## ✨ Features

- User Registration and Login
- Property Search by City
- Property Details Page
- Interested Properties Feature
- Property Filtering and Sorting
- User Dashboard
- Edit Profile Functionality
- Responsive User Interface
- MySQL Database Integration
- REST API Based Operations

- Import the database into phpMyAdmin

- Open the project in browser
